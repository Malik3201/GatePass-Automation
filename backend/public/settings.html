● Local system —

IP mode uses stream and snapshot URLs from LAN cameras. USB mode uses a camera connected to the operator PC through the browser. USB mode works best on http://localhost:5000. If you open the app from another PC using a LAN IP, the browser may require HTTPS before it allows USB camera access. For mobile-as-IP-camera testing, use IP mode with your phone’s IP Webcam URLs. Visitor and CNIC cameras can each be IP or USB independently.
